# Distributed Train Ticket Booking System

## Setup for Windows:
- Run setup.ps1
## Setup for Linux:
- Run setup.sh

Run compile_proto.ps1 / compile_proto.sh after each changes to proto file

Server: 
`python -m src/server/main.py`

Client: 
`python -m src/client/main.py`
